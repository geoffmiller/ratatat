Project at a glance
-------------------
Ratatat is an Ink-compatible React TUI library with a Rust diff engine.

Two ways to use Ratatat
-----------------------
1) React mode (Ink-style components/hooks)
- Use `render()`, `<Box>`, `<Text>`, hooks, etc.
- Main examples: `./examples/`

2) Raw-buffer mode (no React, no Yoga)
- Paint directly into the `Uint32Array` buffer.
- Main examples: `./examples-raw/`
- API details: `./docs/raw-buffer.md`

Basic usage
-----------
Install deps:
- npm install

Build:
- npm run build:ts
- npm run build

Test:
- npm test

Run examples:
- React mode: node --import @oxc-node/core/register examples/kitchen-sink.tsx
- React mode: node --import @oxc-node/core/register examples/use-paste.tsx
- Raw-buffer mode: node --import @oxc-node/core/register examples-raw/conway.ts
